## ⏳ Beschrijving

Een CRUD (Create, Read, Update, Delete) form, waarbij je members van de TribeAPI kan aanpassen en een nieuwe member kan toevoegen met behulp van HTML, CSS & JavaScript.

[🌐 Live](http://justincrud.student.fdnd.nl/)
